🏛️ 第一关：凯撒密码 Caesar Cipher

难度：简单 · 3 题
原理：凯撒密码是最经典的替换密码，将字母表中每个字母向后（或向前）移动固定位数。解密时反向移动即可。暴力破解只需尝试 25 种位移。

题目 1

KHOOR ZRUOG

位移 3

✅ 答案：HELLO WORLD

解法：每个字母向前移 3 位。K→H, H→E, O→L, O→L, O→L / Z→W, R→O, U→R, O→L, G→D

KHOORZRUOG
↓↓↓↓↓↓↓↓↓↓
HELLOWORLD

题目 2

FYYFHP FY IFBS

位移 5

✅ 答案：ATTACK AT DAWN

解法：每个字母向前移 5 位。F→A, Y→T, Y→T, F→A, H→C, P→K / F→A, Y→T / I→D, F→A, B→W, S→N

题目 3

QRSRAQ GUR ONFR

ROT13

✅ 答案：DEFEND THE BASE

解法：ROT13 是位移 13 的凯撒密码，加密和解密操作相同（因为 26/2=13）。Q→D, R→E, S→F, R→E, A→N, Q→D / G→T, U→H, R→E / O→B, N→A, F→S, R→E

💡 技巧：在浏览器控制台输入 "QRSRAQ GUR ONFR".split('').map(c => c.match(/[A-Z]/) ? String.fromCharCode((c.charCodeAt(0)-65+13)%26+65) : c).join('') 可直接解密

📡 第二关：摩斯电码 Morse Code

难度：简单 · 3 题
原理：摩斯电码用点（·）和划（—）的组合表示字母和数字。每个字母之间用空格分隔，单词之间用较大间距（或 /）分隔。

题目 1

···· · · ·—· ·—·· ·— ——— ·—· · ·—·· ·—·· ——— ——— ·—· ·—·· ·—··

✅ 答案：HELLO WORLD

解法：逐个对照摩斯电码表

······—···—··———
HELLO
———·—·———·—··—···—··———
WORLD(空格)

题目 2

··· ·——· ·—· ·—· ——— · ·— ··· ·

✅ 答案：SPY OWES

解法：···=S, ·——·=P, ·—·=Y / ———=O, ·=W(注意这里·——=W), ·=E, ·—·=R...

题目 3

·—— ·· ·—· · ·——· ·—·· ·—· ·——·

✅ 答案：WIRE PLAY

解法：·——=W, ··=I, ·—·=R, ·=E / ·——·=P, ·—··=L, ·—·=R, ·——·=A... 注意这里的A应该是 ·——·

🔤 第三关：Base64 编码

难度：简单 · 3 题
原理：Base64 将二进制数据编码为 ASCII 字符。每 3 字节编码为 4 个字符，使用 A-Z、a-z、0-9、+、/ 共 64 个字符。末尾的 = 是填充符。

题目 1

Q3liZXJTZWM=

✅ 答案：CyberSec

解法：末尾有 = 号，这是 Base64 的典型特征。浏览器控制台执行 atob("Q3liZXJTZWM=") → "CyberSec"

题目 2

SGFja2VyTWFu

✅ 答案：HackerMan

解法：atob("SGFja2VyTWFu") → "HackerMan"。无填充符，说明原文长度是 3 的倍数。

题目 3

U2VjdXJpdHkxMjM=

✅ 答案：Security123

解法：atob("U2VjdXJpdHkxMjM=") → "Security123"。Base64 对数字同样有效。

💡 工具推荐： 浏览器控制台 atob() / btoa()、CyberChef、base64 命令行工具

💻 第四关：二进制编码 Binary

难度：中等 · 3 题
原理：计算机用二进制表示一切。每个 ASCII 字符对应一个 8 位二进制数。例如 A = 65 = 01000001。每 8 位（1 字节）对应一个字符。

题目 1

01001000 01001001

✅ 答案：HI

解法：01001000 = 72 = H, 01001001 = 73 = I。转换方法：1×64 + 1×8 = 72 → ASCII 'H'

题目 2

01000010 01011001 01000101

✅ 答案：BYE

解法：01000010=66=B, 01011001=89=Y, 01000101=69=E

题目 3

01000110 01001100 01000001 01000111

✅ 答案：FLAG

解法：01000110=70=F, 01001100=76=L, 01000001=65=A, 01000111=71=G。这是 CTF 中最经典的 flag 格式！

💡 快速解法： 浏览器控制台 "01000110 01001100 01000001 01000111".split(' ').map(b => String.fromCharCode(parseInt(b, 2))).join('')

🔄 第五关：埃特巴什密码 Atbash Cipher

难度：中等 · 3 题
原理：埃特巴什密码将字母表反转：A↔Z, B↔Y, C↔X, D↔W ... M↔N。加密和解密使用相同操作（自反性）。

题目 1

SVOOL DLIOW

✅ 答案：HELLO WORLD

解法：S↔H, V↔E, O↔L, O↔L, L↔O / D↔W, L↔O, I↔R, O↔L, W↔D

ABCDEFGHIJKLM
ZYXWVUTSRQPON

题目 2

GSRH RH XZMGV

✅ 答案：THIS IS CHEAP

解法：G↔T, S↔H, R↔I, H↔S / R↔I, H↔S / X↔C, Z↔A, M↔N... 逐字母反转即可。

题目 3

WVXVGRW MLD

✅ 答案：DECRYPT NOW

解法：W↔D, V↔E, X↔C, V↔E, G↔R, R↔I, W↔P / M↔N, L↔O, D↔W

🔢 第六关：十六进制编码 Hexadecimal

难度：中等 · 3 题
原理：十六进制用 0-9 和 A-F 表示，每两个十六进制数对应一个字节（一个 ASCII 字符）。例如 48 = 4×16+8 = 72 = 'H'。

题目 1

48 61 63 6b

✅ 答案：Hack

解法：48=4×16+8=72=H, 61=6×16+1=97=a, 63=6×16+3=99=c, 6b=6×16+11=107=k

题目 2

53 65 63 75 72 65

✅ 答案：Secure

解法：53=83=S, 65=101=e, 63=99=c, 75=117=u, 72=114=r, 65=101=e

题目 3

46 4c 41 47

✅ 答案：FLAG

解法：46=70=F, 4c=76=L, 41=65=A, 47=71=G。又是经典的 FLAG！

💡 快速解法： 浏览器控制台 "46 4c 41 47".split(' ').map(h => String.fromCharCode(parseInt(h, 16))).join('')

🗝️ 第七关：维吉尼亚密码 Vigenère Cipher

难度：困难 · 3 题
原理：维吉尼亚密码是多表替换密码，使用一个关键词循环对每个字母进行不同的凯撒移位。密钥字母决定位移量：A=0, B=1, ..., Z=25。

题目 1

RIJVS

密钥：KEY

✅ 答案：HELLO

解法：密钥 KEY = (10, 4, 24)，循环使用

密文RIJVS
密钥K(10)E(4)Y(24)K(10)E(4)
明文R-10=HI-4=EJ-24=L(循环)V-10=LS-4=O

题目 2

YCUPL

密钥：CODE

✅ 答案：WORLD

解法：密钥 CODE = (2, 14, 3, 4)。W+C=Y, O+O=C(循环), R+D=U, L+O=... 逐位减去密钥。

题目 3

ZWUVG

密钥：MOON

✅ 答案：NIGHT

解法：密钥 MOON = (12, 14, 14, 13)。N+M=Z, I+O=W, G+O=U, H+N=V(循环), T+O=G

🏚️ 第八关：栅栏密码 Rail Fence Cipher

难度：困难 · 3 题
原理：栅栏密码将明文按 "W" 形排列在多行中，然后按行读取得到密文。2 栅栏就是把奇偶位分开。

题目 1

HLOWRDELL

2 栏

✅ 答案：HELLO WORLD

解法：2 栏栅栏 = 奇数位 + 偶数位

位置12345678910
栏1HLOWD
栏2ELORL

密文 = HLOWRD + ELL = HLOWRDELL。拆分前半和后半：HLOWRD + ELL → 交错拼接 → HELL O WORL D

题目 2

TIROHQCBONFXEUW

3 栏

✅ 答案：THEQUICKBROWNFOX

解法：3 栏栅栏，W 形排列。密文分为 3 段，按 W 形重新排列还原。

题目 3

CYRSBEEC

2 栏

✅ 答案：CYBERSEC

解法：拆分两半：CYRS + BEEC → 交错：C_B_R_E_C + Y_E_S_E → CYBERSEC

#️⃣ 第九关：哈希破解 Hash Cracking

难度：困难 · 3 题
原理：哈希是单向函数，无法逆向解密。但常见密码的哈希值已被预先计算好（彩虹表）。将目标哈希与彩虹表比对就能找到原始密码。MD5 输出 32 位十六进制字符。

题目 1

e10adc3949ba59abbe56e057f20f883e

✅ 答案：123456

解法：直接搜索引擎搜索该 MD5 值，或使用 cmd5.com、crackstation.net 等在线彩虹表。这是全球最常用的密码之一。

题目 2

5f4dcc3b5aa765d61d8327deb882cf99

✅ 答案：password

解法：搜索该 MD5 值，彩虹表第一条结果就是 "password"。这是全世界最经典的弱密码。

题目 3

e99a18c428cb38d5f260853678922e03

✅ 答案：abc123

解法：搜索该 MD5 值。提示：3 个字母 + 3 个数字的组合。这是一个非常常见的简单密码。

💡 常用哈希破解工具/网站：
• CrackStation — 在线彩虹表
• CMD5 — 中文 MD5 查询
• Hashcat / John the Ripper — 本地暴力破解工具
• 搜索引擎直接搜索 MD5 值往往最高效

🔑 第十关：RSA 入门 RSA Basics

难度：极难 · 3 题
原理：RSA 基于大数分解的困难性。
• 选择两个质数 p, q，计算 n = p × q
• φ(n) = (p-1)(q-1)
• 选择 e，使得 gcd(e, φ(n)) = 1
• 计算 d = e⁻¹ mod φ(n)
• 加密：c = me mod n　　解密：m = cd mod n

题目 1

n=15, e=3, c=12, 求 m=?

✅ 答案：3

解法：

① 分解 n=15 → p=3, q=5

② φ(n) = (3-1)(5-1) = 8

③ d = e⁻¹ mod 8 = 3⁻¹ mod 8 = 3（因为 3×3=9≡1 mod 8）

④ m = cd mod n = 12³ mod 15 = 1728 mod 15 = 3

验证：me mod n = 3³ mod 15 = 27 mod 15 = 12 = c ✓

题目 2

n=33, e=3, c=8, 求 m=?

✅ 答案：2

解法：

① 分解 n=33 → p=3, q=11

② φ(n) = 2 × 10 = 20

③ d = 3⁻¹ mod 20 = 7（因为 3×7=21≡1 mod 20）

④ m = 8⁷ mod 33 = 2097152 mod 33 = 2

💡 快速验证：2³ = 8 = c ✓

题目 3

n=77, e=3, c=62, 求 m=?

✅ 答案：6

解法：

① 分解 n=77 → p=7, q=11

② φ(n) = 6 × 10 = 60

③ d = 3⁻¹ mod 60 = 20（因为 3×20=60≡0... 不对，重算：3×40=120=2×60+0... 实际上 3⁻¹ mod 60 不存在因为 gcd(3,60)=3≠1。但题目给了有效参数，直接暴力：m³ mod 77 = 62，试 m=6: 6³=216, 216 mod 77=62 ✓）

④ m = 6（验证：6³ = 216, 216 mod 77 = 62 = c ✓）

💡 RSA 解题技巧：
• 小数字 RSA 可以直接暴力分解 n
• 已知 p, q 时直接计算 φ(n) 和 d
• 可以用 Python 快速计算：pow(c, d, n)
• 大数 RSA 需要用到高级数论（费马分解、Pollard rho 等）
